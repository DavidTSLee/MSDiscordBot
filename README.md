# MSDiscordBot
Discord bot for Maplestory Reboot Server (Personal Use)


Features:
1) Shows time in multiple time zone (ES(D)T/PS(D)T/KST/AEST)
2) Display server time
3) Ping guild users prior to flag race
4) Ping users prior to reset
